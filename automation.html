Connect your MCP client

Wire up your AI agent to drive this space over Model Context Protocol. Pick your client, paste the config, restart.

{
  "mcpServers": {
    "webfuse-session": {
      "type": "http",
      "url": "https://session-mcp.webfu.se/mcp",
      "headers": {
        "Authorization": "Bearer <your-space-automation-key>"
      }
    }
  }
}

Add to claude_desktop_config.json, then restart Claude.

Add to .cursor/mcp.json in your project root.

{
  "servers": {
    "webfuse-session": {
      "type": "http",
      "url": "https://session-mcp.webfu.se/mcp",
      "headers": {
        "Authorization": "Bearer ${input:automation_key}"
      }
    }
  },
  "inputs": [
    {
      "type": "promptString",
      "id": "automation_key",
      "description": "Space Automation API Key",
      "password": true
    }
  ]
}

Add to .vscode/mcp.json in your workspace, or to user settings.
